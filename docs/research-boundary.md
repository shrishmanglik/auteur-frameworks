# Research and Publication Boundary

## Public

- Generalized framework definitions
- Original schemas and source code
- Aggregate methodology and exact bounded validation denominator
- Synthetic examples
- Deterministic tests and QC rules
- Sanitized synthetic provider-smoke manifests without raw prompts, media, account identities, or provider asset IDs
- Contribution and evidence standards

## Private

- Raw prompts
- Generated image, video, and audio assets
- Per-output extraction records
- Internal record identifiers
- Prompt-to-output mappings
- Private source archives and file paths

## Validation statement

The framework was developed from private longitudinal research and validated against 6,069 rendered outputs from a 6,086-record corpus. Seventeen source records did not contain a corresponding render file and were excluded.

This statement describes the evidence base. It is not a claim that every rule succeeds on every provider, model, content type, or future version.

## Evidence classes

- PROMPT_CORPUS: repeatedly observed in private prompt structures.
- RENDER_VALIDATED: supported by observed output behavior.
- COMBINED: supported on both sides.

These labels expose the kind of support, not private evidence.
