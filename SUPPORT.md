# Support

## Use the right channel

- **Bug or unexpected behavior**: open a bug report with a minimal synthetic reproduction.
- **Framework idea**: open a framework proposal.
- **Integration question**: open a discussion or issue describing the host tool, Node version, command, and observed output.
- **Security or private-data exposure**: follow [SECURITY.md](SECURITY.md) and report privately.

## Include this information

```bash
node --version
npx auteur-frameworks version
npx auteur-frameworks validate your-packet.json
```

Also include the operating system, installation method, full command, exit code, and sanitized error output.

Do not attach private prompts, client media, provider credentials, cookies, account identifiers, or internal source paths.

## Scope

Community support covers the public package, schema, CLI, fixtures, and documentation. Provider account access, billing, generation quality guarantees, commercial integrations, and private AUTEUR OS support are outside this repository's public support commitment.
